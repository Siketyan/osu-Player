# osu! Player
osu! Playerは、リズムゲーム「osu!」の譜面に使用されている楽曲を一覧表示して、再生できるソフトウェアです。

## 動作環境
- Windows 7 以降
- .NET Framework 4.5 がインストールされていること

## 使い方
Zipファイル内の README.txt をご覧ください。
